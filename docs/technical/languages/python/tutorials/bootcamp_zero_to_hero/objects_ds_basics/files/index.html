    Files

    • Interacting with external files
    • we will handle .txt files

    • of we want to handle .docx , .xlsx

    • we can open a file using open(‘path’)

    • we can read the file by a.read()

      a = open('test.txt)
a.read()

    • if you do a read again you will get an empty string , because it read the entire file and put the cursor at the end of file

    • so we have to reset the cursor , to do that we can have a.seek(cursorindex)

    • we can read the lines of the file by a.readlines() , which will give list of element based on new line i.e splitting a string based on new line

    • we have magic function in jupyte for writing files

    %%writefile new.txt
